# PRD — Página SEO: Sistema para Vender Passagens

## Ação Desejada

Criar a página pública `/sistema-para-vender-passagens`, focada em atrair usuários que já estão buscando uma plataforma para vender passagens e converter esse tráfego em novos usuários do SmartBus BR.

---

## Contexto

O SmartBus BR permite:

- criar eventos e viagens
- vender passagens online
- gerar link próprio
- receber pagamentos
- organizar embarque

Essa página deve atuar como página de decisão, ajudando o usuário a escolher um sistema.

---

## Público-alvo

- organizadores de excursões
- empresas de transporte
- vendedores independentes
- operadores que já vendem manualmente

---

## Problema principal

O usuário quer:

- vender passagens online
- organizar melhor sua operação
- ter controle sobre vendas
- profissionalizar o processo

Mas não sabe qual sistema usar.

---

## Proposta da página

Mostrar que:

👉 o SmartBus é uma solução completa  
👉 é simples de usar  
👉 não exige estrutura complexa  
👉 não tem custo fixo
