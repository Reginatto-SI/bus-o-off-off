# Documento Master – Como criar uma conta de empresa no Smartbus BR

## 1. Objetivo do Processo
Este processo tem como objetivo habilitar o primeiro acesso da empresa ao Smartbus BR, criando a conta administrativa inicial para começar a operação comercial e operacional na plataforma.

## 2. Quando Utilizar
Utilize este fluxo quando:
- A empresa ainda não possui cadastro no Smartbus BR.
- A operação está iniciando a venda de passagens/eventos pela primeira vez.
- É necessário formalizar o responsável principal da conta.

## 3. Pré-requisitos
Antes de iniciar, tenha em mãos:
- Definição do tipo de cadastro: Pessoa Jurídica (CNPJ) ou Pessoa Física (CPF).
- Dados da empresa e do responsável.
- E-mail válido (será usado para login).
- Telefone de contato.
- Senha de acesso.
- Conexão com internet e acesso ao navegador em desktop.

## 4. Visão Geral do Processo
A criação da conta é feita pela tela pública de cadastro da empresa. O usuário informa os dados obrigatórios, o sistema valida o preenchimento e, quando aprovado, cria a conta e autentica automaticamente o responsável, direcionando para o painel administrativo.

## 5. Passo a Passo Completo
1. **Acessar o cadastro de empresa**
   - **O que o usuário deve fazer:** abrir a página de cadastro da empresa no Smartbus BR.
   - **O que o sistema faz automaticamente:** apresenta o formulário completo com opções PF/PJ.
   - **O que deve ser conferido:** se a tela de cadastro está carregada corretamente.

2. **Selecionar o tipo de cadastro (PF ou PJ)**
   - **O que o usuário deve fazer:** escolher “Pessoa Jurídica (CNPJ)” ou “Pessoa Física (CPF)”.
   - **O que o sistema faz automaticamente:** ajusta os campos obrigatórios conforme o tipo escolhido.
   - **O que deve ser conferido:** se os campos exibidos correspondem ao tipo selecionado.

3. **Preencher dados gerais da conta**
   - **O que o usuário deve fazer:** informar nome de exibição, nome do responsável, e-mail, telefone, senha e confirmação de senha.
   - **O que o sistema faz automaticamente:** aplica validações de formato (e-mail, senha mínima, confirmação de senha).
   - **O que deve ser conferido:** se não há mensagens de erro no formulário.

4. **Preencher os dados documentais conforme tipo**
   - **O que o usuário deve fazer:**
     - PJ: preencher Razão Social, Nome Fantasia e CNPJ.
     - PF: preencher CPF.
   - **O que o sistema faz automaticamente:** valida CPF/CNPJ e normaliza o documento para cadastro.
   - **O que deve ser conferido:** se o documento foi aceito sem alerta de inválido.

5. **Enviar o cadastro**
   - **O que o usuário deve fazer:** clicar no botão de criação da conta.
   - **O que o sistema faz automaticamente:** cria a empresa com o usuário responsável e tenta autenticação imediata.
   - **O que deve ser conferido:** mensagem de sucesso e redirecionamento para a área administrativa.

6. **Confirmar acesso ao administrativo**
   - **O que o usuário deve fazer:** verificar se foi direcionado para a área de eventos/admin.
   - **O que o sistema faz automaticamente:** inicia sessão com o usuário recém-cadastrado.
   - **O que deve ser conferido:** menu administrativo disponível e conta logada corretamente.

## 6. Pontos de Atenção
- CPF/CNPJ inválido bloqueia o envio.
- E-mail inválido ou senha muito curta impede a criação.
- Divergência entre senha e confirmação interrompe o processo.
- Se ocorrer falha no login automático, a conta pode ter sido criada e o acesso deve ser feito manualmente.

## 7. Boas Práticas
- Usar e-mail corporativo (não pessoal) para a conta principal.
- Definir senha forte e segura já no primeiro acesso.
- Preencher documentos sem abreviações incorretas.
- Revisar os dados antes de concluir para evitar retrabalho posterior.

## 8. Impacto no Sistema
Este fluxo habilita a empresa dentro do Smartbus BR e libera os próximos processos: configuração de dados, conexão de pagamentos, criação de eventos e vendas. Sem conta criada, não há operação comercial, nem gestão de capacidade, nem movimentação financeira no sistema.
