# Documento Master – Como configurar a empresa (dados gerais e identidade visual)

## 1. Objetivo do Processo
Garantir que a empresa esteja corretamente configurada no Smartbus BR com dados institucionais, contato, endereço, vitrine pública e identidade visual.

## 2. Quando Utilizar
Utilize este fluxo quando:
- A conta foi criada e precisa ser preparada para operação.
- Houver mudança de dados cadastrais da empresa.
- Houver atualização de marca (logo, cores, presença pública).

## 3. Pré-requisitos
- Usuário autenticado no painel administrativo.
- Permissão para editar dados da empresa.
- Empresa ativa selecionada na conta.
- Dados oficiais da empresa (nome, documento, contatos e endereço).
- Arquivo de logo em formato aceito (PNG/JPG/SVG).

## 4. Visão Geral do Processo
A configuração é feita na área **Empresa** do administrativo, organizada por abas. O usuário atualiza dados gerais, endereço, contato, observações, identidade visual e define o link público (nick). Ao salvar, o sistema aplica e persiste as alterações da empresa ativa.

## 5. Passo a Passo Completo
1. **Acessar o menu Empresa**
   - **O que o usuário deve fazer:** abrir o menu administrativo e entrar em “Empresa”.
   - **O que o sistema faz automaticamente:** carrega os dados atuais da empresa ativa.
   - **O que deve ser conferido:** se os campos aparecem preenchidos com os dados existentes.

2. **Atualizar Dados Gerais**
   - **O que o usuário deve fazer:** ajustar tipo de cadastro (se aplicável), nomes da empresa e documento.
   - **O que o sistema faz automaticamente:** valida CPF/CNPJ e padroniza os dados para gravação.
   - **O que deve ser conferido:** se não há alertas de validação antes de salvar.

3. **Configurar vitrine pública (nick/link curto)**
   - **O que o usuário deve fazer:** informar o nick da empresa para link público.
   - **O que o sistema faz automaticamente:** valida formato e disponibilidade do nick.
   - **O que deve ser conferido:** indicador de nick disponível antes de confirmar.

4. **Atualizar Endereço, Contato e Observações**
   - **O que o usuário deve fazer:** preencher ou revisar endereço comercial, telefone e demais contatos.
   - **O que o sistema faz automaticamente:** registra os dados na empresa ativa.
   - **O que deve ser conferido:** consistência dos canais de contato para clientes e equipe.

5. **Atualizar identidade visual**
   - **O que o usuário deve fazer:**
     - Enviar ou trocar a logo.
     - Ajustar cores institucionais (onde disponível).
   - **O que o sistema faz automaticamente:** envia a logo para armazenamento e vincula a URL na empresa.
   - **O que deve ser conferido:** se a prévia da marca foi atualizada corretamente.

6. **Salvar alterações**
   - **O que o usuário deve fazer:** clicar em “Salvar alterações”.
   - **O que o sistema faz automaticamente:** grava os dados e atualiza o contexto da empresa ativa.
   - **O que deve ser conferido:** confirmação de sucesso e permanência dos dados após atualização da tela.

## 6. Pontos de Atenção
- Não é recomendado alterar tipo jurídico sem revisar todos os campos relacionados.
- Nick reservado ou já utilizado não pode ser salvo.
- Upload de logo fora do formato/tamanho permitido será rejeitado.
- Evite deixar contatos desatualizados para não prejudicar atendimento.

## 7. Boas Práticas
- Padronizar nome público e nome legal conforme documentos oficiais.
- Definir logo com boa legibilidade em fundo claro e escuro.
- Manter endereço e telefone sempre atualizados.
- Estabelecer revisão periódica dos dados institucionais.

## 8. Impacto no Sistema
A configuração da empresa impacta diretamente a comunicação com clientes, a apresentação da marca na vitrine pública e a consistência administrativa. Também influencia confiança comercial, eficiência operacional e qualidade de atendimento pós-venda.
