# Documento Master – Como cadastrar um motorista

## 1. Objetivo do Processo
Cadastrar motoristas para vinculação nas viagens dos eventos, garantindo organização operacional, rastreabilidade e controle de disponibilidade.

## 2. Quando Utilizar
Utilize este fluxo quando:
- Um novo motorista passa a atuar na empresa.
- É necessário atualizar dados de condutor existente.
- Houver necessidade de ativar/inativar profissionais no sistema.

## 3. Pré-requisitos
- Acesso administrativo ativo.
- Empresa ativa selecionada.
- Permissão para gestão de motoristas.
- Dados pessoais e profissionais do motorista (ex.: CNH e contatos).

## 4. Visão Geral do Processo
O processo ocorre na tela **Motoristas**, com listagem e formulário em abas. O usuário cadastra dados pessoais, documentos e informações de habilitação, salva o registro e controla o status operacional (ativo/inativo).

## 5. Passo a Passo Completo
1. **Acessar a tela de Motoristas**
   - **O que o usuário deve fazer:** abrir o menu “Motoristas”.
   - **O que o sistema faz automaticamente:** carrega os registros existentes e filtros.
   - **O que deve ser conferido:** se já existe cadastro do profissional.

2. **Iniciar novo cadastro**
   - **O que o usuário deve fazer:** clicar em “Adicionar Motorista”.
   - **O que o sistema faz automaticamente:** abre o formulário para inclusão.
   - **O que deve ser conferido:** se o formulário está no modo de criação.

3. **Preencher dados principais**
   - **O que o usuário deve fazer:** inserir nome, documentos, contatos e informações de habilitação.
   - **O que o sistema faz automaticamente:** valida formatação e campos obrigatórios.
   - **O que deve ser conferido:** se CNH/categoria e dados de contato estão corretos.

4. **Salvar e confirmar cadastro**
   - **O que o usuário deve fazer:** concluir com o botão de salvar.
   - **O que o sistema faz automaticamente:** grava o motorista e atualiza a listagem.
   - **O que deve ser conferido:** se o motorista aparece na tabela com status adequado.

5. **Ajustar status operacional**
   - **O que o usuário deve fazer:** ativar ou desativar conforme disponibilidade real.
   - **O que o sistema faz automaticamente:** alterna o status do motorista no cadastro.
   - **O que deve ser conferido:** se o status reflete a escala operacional atual.

## 6. Pontos de Atenção
- Evitar duplicidade de cadastro para o mesmo motorista.
- Revisar validade e categoria da CNH antes de vincular em viagens.
- Manter contato atualizado para comunicação operacional.

## 7. Boas Práticas
- Usar padrão único de preenchimento de nome e telefone.
- Registrar observações relevantes de operação quando necessário.
- Revisar periodicamente motoristas inativos e ativos.

## 8. Impacto no Sistema
Motoristas corretamente cadastrados garantem vínculo correto nas viagens, organização da escala e melhor controle operacional do evento. Isso reduz risco de inconsistências em operação e atendimento no embarque.
